# Правила дизайну сайту

## Загальні принципи
- Використовувати компоненти Text та Heading замість прямого використання HTML тегів
- Дотримуватися системи типографіки та відступів
- Забезпечити консистентність на всіх сторінках

## Типографіка

### Розміри тексту (Text component)
- **xs**: 12px (0.75rem) - для мінімального тексту, міток, моно-тексту
- **sm**: 14px (0.875rem) - для допоміжного тексту, підписів
- **base**: 16px (1rem) - стандартний розмір для основного тексту (за замовчуванням)
- **lg**: 18px (1.125rem) / 20px (1.25rem) на md+ - для підзаголовків, акцентного тексту
- **xl**: 20px (1.25rem) / 24px (1.5rem) на md+ - для великих підзаголовків
- **2xl**: 24px (1.5rem) / 30px (1.875rem) на md+ - для секційних заголовків
- **3xl+**: тільки для спеціальних випадків

### Заголовки (Heading component)
- **level 1**: text-4xl md:text-5xl lg:text-6xl - головні заголовки сторінок
- **level 2**: text-3xl md:text-4xl lg:text-5xl - секційні заголовки
- **level 3**: text-2xl md:text-3xl - підзаголовки в картках
- **level 4-6**: для менших заголовків

### Вага шрифту
- **light**: для великих заголовків (Hero)
- **normal**: стандартна вага для основного тексту
- **medium**: для підзаголовків та акцентного тексту
- **bold**: для заголовків

## Відступи (Spacing)

### Вертикальні відступи між секціями
- **Мінімальний**: py-16 md:py-20 (64px/80px)
- **Стандартний**: py-20 md:py-24 (80px/96px) - використовувати за замовчуванням
- **Великий**: py-24 md:py-32 (96px/128px) - для Hero та CTA секцій

### Відступи всередині секцій
- **Між заголовком секції та контентом**: mb-12 md:mb-16 (48px/64px)
- **Між елементами в списку**: gap-4 (16px) або gap-6 (24px)
- **Між картками**: gap-4 (16px) для малих, gap-6 (24px) для великих
- **Між текстом та кнопками**: mt-8 (32px) або mt-10 (40px)

### Відступи в картках
- **Padding карток**: p-6 md:p-8 (24px/32px)
- **Відступ між заголовком та текстом в картці**: mb-3 (12px) або mb-4 (16px)
- **Відступ між текстом та тегами**: mb-6 (24px)

### Горизонтальні відступи
- **Контейнер padding**: px-6 (24px) - стандарт
- **Між колонками в grid**: gap-6 (24px) або gap-8 (32px)

## Кольори тексту

### Основні
- **foreground** (#EDEDED) - основний текст
- **muted-foreground** (#888888) - другорядний текст
- **white/80** або **white/60** - для тексту на темному фоні

### Акценти
- **accent** (#6DD620) - для акцентних елементів
- **text-zinc-300** або **text-zinc-400** - для підзаголовків

## Компоненти

### Text component
ЗАВЖДИ використовувати компонент Text замість прямого `<p>`, `<span>`:
```tsx
<Text size="base">Основний текст</Text>
<Text size="sm" variant="muted">Допоміжний текст</Text>
<Text variant="mono" size="xs">Моно текст</Text>
```

### Heading component
ЗАВЖДИ використовувати компонент Heading замість прямого `<h1>`, `<h2>`, `<h3>`:
```tsx
<Heading level={1}>Головний заголовок</Heading>
<Heading level={2}>Секційний заголовок</Heading>
<Heading level={3}>Підзаголовок</Heading>
```

### SectionHeader
Використовувати для всіх секційних заголовків з описом.

## Заборонено

1. ❌ Пряме використання `text-14px`, `text-16px` - використовувати size props
2. ❌ Пряме використання `<p className="text-base">` - використовувати `<Text>`
3. ❌ Пряме використання `<h1>`, `<h2>`, `<h3>` - використовувати `<Heading>`
4. ❌ Неконсистентні відступи (mb-3, mb-4, mb-6 в одній секції) - дотримуватися системи
5. ❌ Різні розміри тексту для одного типу контенту (наприклад, описи карток)

## Приклади правильного використання

### Секція з картками
```tsx
<section className="py-20 md:py-24">
  <div className="max-w-7xl mx-auto px-6">
    <div className="mb-12 md:mb-16">
      <SectionHeader ... />
    </div>
    <div className="grid gap-6">
      <Card className="p-6 md:p-8">
        <Heading level={3} className="mb-4">Заголовок</Heading>
        <Text size="base" className="mb-6">Опис</Text>
      </Card>
    </div>
  </div>
</section>
```

### Текстовий блок
```tsx
<div className="space-y-4">
  <Heading level={2}>Заголовок</Heading>
  <Text size="lg" variant="muted">Підзаголовок</Text>
  <Text size="base">Основний текст</Text>
</div>
```
